
STEPS TO EXECUTE THE PROJECT:

Step 1 : Intially, Using Bluetooth voice note  App connect to HC 05 device
Step 2 : After connecting, to print voice command ,type any medical words in SAY and GET and click add
             Ex: SAY - dolo650 , GET - *dolo650#
Step 3 : The Robot can be adjusted and fixed accordingly using the buttons available in the kit
Step 4 : Now start giving the voice command
Step 5 : Finally the robot starts writing the message which is given through voice command
